App Config

Demo vs. live: Deadlines are real app settings and work everywhere. The faculty password and OAuth ID below are server secrets on a live site — set them as Netlify environment variables (this page shows you exactly what to paste). The values here also drive the offline demo so you can show a gated login.

Faculty dashboard password

Used to gate the Faculty dashboard in the offline demo on this browser.

Google OAuth (sign-in)

For a live site, paste your OAuth Client ID. The email sign-in always works without this; the Google button only fully works once the serverless functions are deployed.

Deadlines

Set a due date per question. Past the deadline, student submissions are flagged Late in the student app and faculty dashboard. Leave blank for no deadline.
